## Overview
母平均が同じ2つの母集団から乱数をたくさん生成して、それらに対して平均値の差の検定(両側)を繰り返します。  
p値を集計し、「有意水準αの検定を実施した時の実際のType I error」がどのようになっているかをグラフにします。
